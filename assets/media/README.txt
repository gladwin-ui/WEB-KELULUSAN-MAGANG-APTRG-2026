Folder Media — APTRG Web Kelulusan
==================================

Berisi aset gambar lokal (tanpa URL eksternal).

File:
  logo-aptrg.png       — Logo resmi APTRG Laboratory (ganti file ini jika ada versi baru)
  bg-tech-pattern.svg  — Background SVG cadangan (opsional)

Background utama web:
  ../background.jpeg   — Foto background halaman (di folder assets/)
  hero-banner.svg      — Banner dekoratif opsional (bisa dipakai di halaman lain)

Cara ganti logo:
  1. Simpan gambar baru dengan nama yang sama (logo-aptrg.png), atau
  2. Ubah path di index.html: src="assets/media/nama-file-anda.png"

Cara ganti background:
  1. Ganti bg-tech-pattern.svg, atau tambahkan bg-foto.jpg di folder ini
  2. Ubah di styles.css → .bg-image { background-image: url('assets/media/...'); }
